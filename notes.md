# Git & GitHub Notes

## What is Git?
Git is a distributed version control system used to track changes in source code.

## What is GitHub?
GitHub is a cloud-based platform that hosts Git repositories and allows collaboration.

## Basic Git Operations
- **Stage**: Select files for commit
- **Commit**: Save changes with a message
- **Push**: Send local commits to remote repository
- **Pull**: Fetch changes from remote repository
